Imputation by Regression

Regression-based imputation is a technique that uses the relationships between features to estimate missing values. This approach is useful when features are correlated, as it allows us to make more informed predictions based on observed data. This method often yields more accurate imputations than simple strategies like using the mean or mode. For numerical features, we use linear regression to predict the missing values. For categorical features, we apply logistic regression.

Imputation Strategy

The correlation matrix shows that Height and Weight are strongly correlated with Gender. In addition, the feature distributions indicate noticeable differences in the mean values of Height and Weight across Gender groups. When imputing missing values for these features, it is therefore sensible to include Gender as a predictor to account for this group-level effect. More generally, Gender may be included in imputation models for other features as well, since subtle group differences might exist even if not immediately apparent.

We will begin imputation with Body_Temp as it has the lowest percentage of missing values. Starting with features that are mostly complete ensures that we use as much observed data as possible when training our imputation models. This helps improve the reliability of the predictions, especially in a sequential imputation strategy where previously imputed values may be used to impute others. Body_Temp is highly correlated with both Duration and Heart_Rate, making either a suitable predictor for regression-based imputation. However, since Duration and Heart_Rate are themselves strongly correlated, including both may introduce redundancy or multicollinearity. Therefore, we will initially use Duration and Gender as the predictor for imputing missing values in Body_Temp. We will use Heart_Rate and Gender as a fallback for any remaining null values and finally impute an average value, grouped by Gender as a second fallback.

For categorical features, the appropriate average is the mode.
In numerical features, if the distribution is symmetric and free of outliers, the mean is a suitable imputation value. If the distribution is skewed or contains outliers, the median provides a more robust estimate.

Body_Temp is heavily left skewed so will use the median for the second fallback imputation.

Height has the next lowest percentage of missing values. This is highly correlated with Weight so we will use Weight and Gender as a predictor for imputation by regression. We will also use a fallback imputation and, since Height is approximately normally distributed we will use the mean grouped on Gender.

Weight has the next lowest percentage of missing values. As we know, this is highly correlated with Height so we will use Height and Gender as a predictor for imputation by regression. We will also use a fallback imputation and, since Weight is also approximately normally distributed we will use the mean grouped on Gender.

Heart_Rate has the next lowest percentage of missing values. We will use Duration (the highest correlated feature), and Gender, as predictors in the first regression imputation, followed by Body_Temp and Gender for a fallback. Heart_Rate is also approximately normally distributed so we will use the mean grouped by Gender for a second fallback.

Duration has the next lowest percentage of missing values. We will use Body Temp (the highest correlated feature), and Gender, as predictors in the first regression imputation, followed by Heart_Rate and Gender for a fallback. Duration is approximately uniformly distributed so we will use the mean grouped by Gender for a second fallback.

Age has the next lowest percentage of missing values. This feature is not significantly correlated with any other features so regression imputation is not appropriate. We could consider imputing an average value but since this is the primary imputation there is a risk that it could intoduce bias in the data (4.2% of the data would be imputed the same value). Instead, to preserve the distribution of the feature, we will use a probabilistic imputer grouped by Gender.

Lastly, Gender has the highest percentage of missing values and is highly correlated with Height and Weight. As Height and Weight are also highly correlated we will select only Weight as predictor. As this will be a binary categorical variable logistic regression imputation would be an appropriate method. We can then use Height as a fallback, and any remaining null values can be imputed probabilistically to preserve the original class distribution (which is approximately 50/50).

In all these imputations we will also add an indicator column to the dataset to record which values have been imputed. It is always possible that there is predictive information in the missingness of values.

class SequentialRegressionImputer:
    def __init__(self, target_col, numerical_predictors, cat_predictor='Gender', 
                 final_strategy='median', strictly_positive=False):
        """
        target_col: Column to impute
        numerical_predictors: Ordered list of numerical predictors to try
        cat_predictor: Categorical predictor column (default 'Gender')
        final_strategy: 'mean' or 'median' for final fallback
        strictly_positive: If we are only imputing values greater than 0
        """
        self.target_col = target_col
        self.numerical_predictors = numerical_predictors
        self.cat_predictor = cat_predictor
        self.final_strategy = final_strategy 
        self.strictly_positive = strictly_positive
               
        # Standard OneHotEncoder configuration: drop='first' avoids multicollinearity,
        # handle_unknown='ignore' prevents errors from unseen categories at inference,
        # sparse_output=True saves memory for high-cardinality features.
        # For binary features, sparse_output=False is usually fine.
        self.encoder = OneHotEncoder(handle_unknown='ignore', drop='first', sparse_output=False)

        
    def fit(self, X, y=None):       
        self.models_ = {}
        # Prepare categorical encoder
        self.encoder.fit(X[[self.cat_predictor]])
        
        # Train models for each predictor
        for num_pred in self.numerical_predictors:               
            train_data = X[[num_pred, self.cat_predictor, self.target_col]].dropna()

            # need a minimum of 2 points
            if len(train_data) > 1:
                X_train = self._prepare_features(train_data, num_pred)
                y_train = train_data[self.target_col]
                
                model = LinearRegression()
                model.fit(X_train, y_train)
                self.models_[num_pred] = model
                
        return self
    
    def transform(self, X):
        check_is_fitted(self)
        X = X.copy()
        
        # Initialize missing mask
        missing_mask = X[self.target_col].isna()
        
        # Add missing indicator before imputation
        indicator_col = f"{self.target_col}_imputed"
        X[indicator_col] = missing_mask.astype(int)
        
        # Skip imputation if no missing values
        if not missing_mask.any():
            return X
            
        # Try predictors in order
        for num_pred in self.numerical_predictors:
            if not missing_mask.any():
                break
                
            if num_pred in self.models_:
                try:   
                    pred_data = X.loc[missing_mask, [num_pred, self.cat_predictor]]            
                    X_pred = self._prepare_features(pred_data, num_pred)
                    preds = self.models_[num_pred].predict(X_pred)
                    
                    # if predictions must be strictly positive
                    if self.strictly_positive:
                        preds = np.clip(preds, a_min=1.0, a_max=None)
                        
                    X.loc[missing_mask, self.target_col] = preds
                    
                    missing_mask = X[self.target_col].isna()
                except:
                    continue
        
        # Final fallback
        if missing_mask.any():
            if self.final_strategy == 'mean':
                fill_value = X[self.target_col].mean()
            else:
                fill_value = X[self.target_col].median()
            X.loc[missing_mask, self.target_col] = fill_value
        
        return X
    
    def fit_transform(self, X, y=None):
        return self.fit(X).transform(X)
    
    def _prepare_features(self, df, numerical_predictor):
        """Prepare feature matrix with encoded categorical"""
        cat_encoded = self.encoder.transform(df[[self.cat_predictor]])
        # returns array of numerical_predictor and One Hot Encoded values
        return np.column_stack([
            df[numerical_predictor].values,
            cat_encoded
        ])
        

class ProbabilisticImputer:
    def __init__(self, feature, groupby=None, random_state=None):
        """
        feature: Column name to impute
        groupby: Column name(s) to group by (optional)
        random_state: For reproducibility
        """
        self.feature = feature
        self.groupby = groupby
        self.random_state = random_state

    def fit(self, X, y=None):
        # reproducible random number generator
        self.rng_ = np.random.RandomState(self.random_state)
        self.distributions_ = {}
        
        if self.groupby is None:
            # Global distribution
            non_null = X[self.feature].dropna()
            if len(non_null) > 0:
                self.distributions_['global'] = non_null.values
        else:
            # Grouped distributions
            groupby_cols = [self.groupby] if isinstance(self.groupby, str) else list(self.groupby)
            for name, group in X.groupby(groupby_cols)[self.feature]:
                non_null = group.dropna()
                if len(non_null) > 0:
                    self.distributions_[name] = non_null.values

        return self

    def transform(self, X):
        check_is_fitted(self)
            
        X = X.copy()
        missing_mask = X[self.feature].isna()
        
        X[f"{self.feature}_imputed"] = missing_mask.astype(int)
        
        if not missing_mask.any() or not self.distributions_:
            return X
            
        if self.groupby is None:
            # Global imputation
            samples = self.rng_.choice(
                self.distributions_['global'],
                size=missing_mask.sum(),
                replace=True
            )
            X.loc[missing_mask, self.feature] = samples
        else:
            # Grouped imputation
            groupby_cols = [self.groupby] if isinstance(self.groupby, str) else list(self.groupby)
            current_groups = X.groupby(groupby_cols).groups
            
            for group_name in current_groups:
                if group_name not in self.distributions_:
                    continue
                    
                if isinstance(self.groupby, str):
                    group_mask = (X[self.groupby] == group_name)
                else:
                    group_mask = (X[groupby_cols] == group_name).all(axis=1)
                
                group_missing = missing_mask & group_mask
                if group_missing.any():
                    X.loc[group_missing, self.feature] = self.rng_.choice(
                        self.distributions_[group_name],
                        size=group_missing.sum(),
                        replace=True
                    )
        return X

    def fit_transform(self, X, y=None):
        return self.fit(X).transform(X)
 

class SequentialLogisticImputer(BaseEstimator, TransformerMixin):
    def __init__(self, target_col, predictors, random_state=None):
        self.target_col = target_col
        self.predictors = predictors  # List of predictors 
        self.random_state = random_state

    def fit(self, X, y=None):
        # Save observed class probabilities for fallback
        self.models_ = {}
        observed = X[self.target_col].dropna()
        self.class_probs_ = observed.value_counts(normalize=True).to_dict()

        # Fit one model per predictor
        for predictor in self.predictors:
            mask = X[self.target_col].notna() & X[predictor].notna()
            X_train = X.loc[mask, predictor].values.reshape(-1, 1)
            y_train = X.loc[mask, self.target_col]
            model = LogisticRegression(random_state=self.random_state)
            model.fit(X_train, y_train)
            self.models_[predictor] = model

        return self

    def transform(self, X):
        check_is_fitted(self)
        X = X.copy()
        missing_mask = X[self.target_col].isna()

        # Indicator column
        X[f"{self.target_col}_imputed"] = missing_mask.astype(int)

        for predictor in self.predictors:
            mask = X[self.target_col].isna() & X[predictor].notna()
            if mask.any():
                X_pred = X.loc[mask, predictor].values.reshape(-1, 1)
                preds = self.models_[predictor].predict(X_pred)
                X.loc[mask, self.target_col] = preds

        # Final fallback: probabilistic imputation
        final_mask = X[self.target_col].isna()
        if final_mask.any() and self.class_probs_:
            values = np.random.choice(
                list(self.class_probs_.keys()),
                size=final_mask.sum(),
                p=list(self.class_probs_.values())
            )
            X.loc[final_mask, self.target_col] = values

        return X
